Figured Arpeggio v2

A Philip Glass-style arpeggiator implementing authentic minimalist techniques: additive process, stanzaic form, defective time loops, and parallel motion. Designed for MIDI Fighter Twister (16 knobs per page).

Row 1: Core Musical Parameters

1. TempoMaster clock speed (40-200 BPM)
2. RootTonic note of the scale (C through B)
3. ModeScale type: Minor, Major, Dorian, Phrygian, Lydian, Mixolydian, Aeolian
4. Cell SizeNumber of notes in the base melodic cell (2-8). This is the "generator" pattern that gets expanded.

Row 2: Additive Process

The core Glass technique from "Two Pages" and "Music in Fifths" - adding new elements to a basic melodic structure.

5. ExpandHow many notes to add to the base cell (0-8). New notes are derived from existing ones.
6. GrowthHow expansion evolves over time:
  • Static - No change
  • Linear ↑ - Gradually adds notes (1→2→3→4)
  • Linear ↓ - Gradually removes notes (4→3→2)
  • Pendulum - Grows then shrinks (1→2→3→4→3→2→1...)
  • Glass Irregular - Forward 2, back 1 (mimics "Two Pages")
  • Defective Loop - Random omissions creating the famous "defective time loop"
7. SubdivRhythmic subdivision: 8th notes, 16ths, triplets, or quintuplets
8. GateNote length as fraction of step (0.1-1.0). Lower = staccato, higher = legato

Row 3: Texture & Form

9. TextureSecond voice relationship (all use diatonic intervals):
  • Mono - Single voice
  • Parallel 3rds - Diatonic thirds (minor/major depends on mode)
  • Parallel 4ths - Diatonic fourths
  • Parallel 5ths - Diatonic fifths (like "Music in Fifths")
  • Parallel 6ths - Diatonic sixths
  • Contrary - Mirror motion (voice 2 moves opposite direction in scale degrees)
  • Bicinium - Same pattern one octave below
10. CanonOffset in steps for voice 2 (0-8). Creates canon/round effects.
11. HarmonicChord progression (all use scale degrees, work in any mode):

Glass Signatures

  • Koyaanisqatsi - i-VII-VI-VII oscillation
  • Einstein - i-V-i (Knee Plays)
  • Akhnaten - Descending: i-VII-VI-V-IV-III
  • Metamorphosis - Cycling thirds: i-III-V-III
  • Glassworks - Pendulum: i-V-IV-V

Reich Signatures

  • Music for 18 - Pulsing: i-IV-i-V
  • Different Trains - Minor second movement: i-II-i-VII

Voice Leading

  • Step Up - Ascending stepwise: i-II-III-IV
  • Step Down - Descending lament: i-VII-VI-V
  • Thirds - Ascending thirds: i-III-V-VII
  • Fourths - Quartal movement: i-IV-VII-III
  • Fifths - Circle segment: i-V-II-VI

Modal

  • i-IV Shuttle - Classic Dorian sound
  • Aeolian - i-VI-VII cadence
  • Phrygian - i-bII half-cadence
12. StanzaForm position (1-6). Affects intensity/velocity: 1-2 building, 3-4 climax, 5-6 resolution. Also shifts entry point in harmonic progression.

Row 4: Character & Output

13. FilterLowpass filter cutoff (20-20000 Hz)
14. ResFilter resonance (0-1)
15. RegisterTranspose by scale degrees (-24 to +24). +7 = one octave up in a 7-note scale. Stays diatonic.
16. DensityProbability that each note plays (0-1). At 1.0 all notes play. Lower values create rests - the "defective time loop" effect. First/last notes always play.

Synth Parameters

Expand the collapsible sections below the Twister grid to access full Molly-style synth parameters:

  • Oscillator - Waveform (Tri/Saw/Pulse), pulse width modulation, frequency modulation, glide
  • Mixer - Main oscillator, sub oscillator, noise levels
  • Filter - High-pass, low-pass with envelope modulation and keyboard tracking
  • Envelopes - Two ADSR envelopes for amp and modulation
  • LFO - Multiple waveforms with fade-in control
  • Ring Mod - Ring modulation with fade and mix
  • Chorus - Stereo chorus effect

Preset Buttons

  • Create Lead - Bright, cutting lead sounds
  • Create Pad - Soft, evolving pad textures
  • Create Percussion - Plucky, percussive tones

Tips

  • Start with Cell Size: 4, Expand: 0, then slowly increase Expand to hear the additive process
  • Use Growth: Glass Irregular for authentic "Two Pages" feel
  • Combine Texture: Parallel 5ths with Canon: 2 for rich counterpoint
  • Lower Density to 0.7-0.8 for subtle rhythmic variation
  • All notes stay strictly within the chosen scale - no sour notes!
