Noise Merchant

A generative noise synthesizer featuring dual formant oscillators with self-modulating feedback through a state-variable filter (SVF).

Signal Flow

Two formant oscillators with cross-modulation feed into a Sieve SVF filter. The filter's output is fed back to modulate the oscillators' pitch and frequency, creating complex, evolving timbres.

Controls

  • Pitch 1/2: Base pitch for each oscillator in V/Oct (-2 to +2 octaves)
  • Formant 1/2: Formant frequency control - affects harmonic structure
  • Offset: Frequency offset multiplier applied via feedback
  • Barrel: Waveshaping amount - adds harmonics and edge
  • Filter: Filter cutoff in V/Oct
  • Fil Offset: Filter frequency offset (fine control)
  • Q: Filter resonance
  • Mix: Dry/wet mix for filter
  • Drive: Output saturation amount

Low Pass Gate (Optional)

Toggle the LPG to add plucky, organic amplitude shaping:

  • Attack: How fast the gate opens (1-500ms)
  • Release: Decay time of the envelope (50-2000ms)
  • Mod Depth: How much the envelope affects filter cutoff

Tips

  • Set Pitch 1 and Pitch 2 to different values for beating/interference patterns
  • Increase Barrel for more aggressive, distorted tones
  • Use Filter and Q together to create resonant sweeps
  • The feedback creates self-modulation - small parameter changes can have big effects
  • MIDI keyboard controls Pitch 1, modwheel controls Filter

MIDI Control

  • Notes: Control Pitch 1 (C3 = center)
  • Mod Wheel: Filter cutoff
  • Velocity: Affects barrel/drive

Keyboard Shortcuts

  • Space: Play/Pause
  • R: Randomize parameters
  • Z/X: Pitch 1 down/up
  • A/S: Pitch 2 down/up
  • T: Trigger LPG (when enabled)
