OTA Rev 3

An aggressive variant of OTA featuring aliasing sawtooth and pulse oscillators with pulse width modulation, processed through a Sieve state-variable filter.

Changes from Rev 2

  • Aliasing Saw: Osc 1 now uses sawtooth waveform for richer harmonics
  • Aliasing Pulse: Osc 2 uses pulse wave with adjustable pulse width
  • Sieve SVF: State-variable filter replaces Bique for LP/BP/HP morphing
  • SV Mix: Crossfade between filter modes

Oscillator Section

  • Freq 1: Saw oscillator fine tune
  • Vol 1 / Oct 1: Saw volume and octave
  • Freq 2: Pulse oscillator fine tune
  • PW: Pulse width (0.1 = thin, 0.5 = square, 0.9 = wide)
  • Vol 2 / Oct 2: Pulse volume and octave
  • Osc Sync: Hard sync pulse to saw

SVF Filter

  • SV Freq: Filter cutoff frequency
  • SV Q: Resonance (self-oscillates near max)
  • SV Gain: Filter output gain
  • SV Mix: 0=Lowpass, 0.5=Bandpass, 1=Highpass

LPG (Low Pass Gate)

  • LPG Rise: Attack time
  • LPG Fall: Decay time
  • LPG Bend: Envelope curve shape
  • LPG Hght: Maximum opening

Autogen

  • BPM: Tempo (40-200)
  • Mul/Div: Clock multiplier/divider
  • Trig Prob: Trigger probability
  • Scale: Pitch quantization scale

MIDI Control

  • Notes: Pitch and trigger
  • Mod Wheel: SV Filter frequency
  • CC 2: Pulse width
  • CC 3: SV Mix (filter type morph)

Keyboard

  • Space: Toggle autogen
  • Z-M: Play notes
  • Q/W: PW down/up
